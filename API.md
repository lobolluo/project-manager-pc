## 通用响应格式

### 成功响应

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

### 错误响应

```json
{
  "code": 40001,
  "message": "错误描述",
  "data": null
}
```

### 错误码定义

| 错误码 | 含义 |
|--------|------|
| 0 | 成功 |
| 40001 | 参数错误 |
| 40101 | 未认证 |
| 40301 | 无权限 |
| 50001 | 服务端错误 |

## 认证模块

### POST /auth/login

登录接口

**请求参数**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| phone | string | 是 | 手机号 |
| password | string | 是 | 密码 |

**响应示例**：

```json
{
  "code": 0,
  "data": {
    "token": "xxx",
    "user": {}
  }
}
```
