# ProjectFlow - 开发规划

## 项目概述

ProjectFlow — 轻量级项目管理工具，面向 5-20 人小团队，提供看板、任务追踪、团队协作等核心功能。

---

## 技术架构

```
Next.js 16 (App Router)
├── TypeScript + Tailwind CSS v4
├── Shadcn UI (12 组件)
├── Zustand (客户端状态) + TanStack Query (服务端状态)
├── @dnd-kit (拖拽)
├── React Hook Form + Zod (表单校验)
└── localStorage (数据持久化)
```
